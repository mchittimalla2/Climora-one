# Climoraone Project Context

## Business
Climoraone is an e-commerce platform focused on eco-friendly handmade products created by rural women artisans.

## Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- Laravel

Future:
- MySQL
- Razorpay
- Email Notifications
- WhatsApp Notifications

## Current Features

Completed:
- Store Page
- Cart
- Checkout
- Admin Dashboard
- Product Management
- Order Management
- Reports
- Track Order Page

Current Data Storage:
- LocalStorage

Next Phase:
- Laravel APIs
- MySQL Database
- Email Notifications

## Coding Standards

- Keep UI simple
- Non-technical admins should be able to manage products
- Mobile first
- Clean React components
